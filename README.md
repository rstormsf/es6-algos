  Problem

## Array Pair sum

  Problem  
  Given an integer array, output all the unique pairs that sum up to a specific value k.  
  So the input:
  `pair_sum([1,3,2,2],4)`
  would return 2 pairs:
  `
 (1,3)
 (2,2)`

**Source**: [pair_sum.js](src/pair_sum.js)  
**Test**: [pair_sum_test.js](test/pair_sum_test.js)
